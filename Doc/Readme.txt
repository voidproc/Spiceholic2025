# Blazing Spiceholics

https://github.com/voidproc/Spiceholic2025

「辛味（からみ）」に取り憑かれ、新たな辛さを求めてさまよう一人の少女がいた。

低解像度でレトロチックな 2D 見下ろし型アクションパズルゲームです。

このゲームは「バンダイナムコスタジオ杯 | Siv3D ゲームジャム 2025」参加作品です。
https://bandainamcostudios.connpass.com/event/364446/

## ゲームジャム 作品紹介ページ
- https://scrapbox.io/bnscup2025/Blazing_Spiceholics

## 操作
移動と攻撃のみのシンプル操作です。キーボードまたはゲームコントローラで操作できます。

### キー・ボタン割り当て（初期設定）
- 移動:           方向キー / 方向ボタン
- 攻撃／決定:     Z キー   / ボタン 1
- ポーズメニュー: Esc キー / ボタン 3

## ルール
- 炎を吐き、敵を倒しながら先に進みましょう。
- とうがらしがあったら、食べましょう。
- ステージ内の敵を全滅させるか、「カギ」を取ると次のステージへ進みます。
